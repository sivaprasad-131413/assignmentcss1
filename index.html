What is Css?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be display on screen,paper or in other media

  3. Css is not necessary Css saves a lot of work It can control the layout of multiple web pages all at once.

  4. External stylesheets are stored in CSS files

CSS can be added to HTML document in3 ways:

  • Inline - by using the sytle attribute inside HTML elements.
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the sytle in external CSS file. However, in this tutorial we will see inline and internal styles, because this is easier to demostrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies s style to multiple elements ID, on the other hand, applies the style to a unique element.ID is also special in that you can use a special URL to link directly to an element and it's used by Javascript.

In CSS,selectors are used to target a specific element or a range of elements on a web page.Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
